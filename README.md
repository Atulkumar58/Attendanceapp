# 📊 Attendance Management System

A **web-based Attendance Management System** developed to help faculty
or administrators efficiently manage and track student attendance.

The system replaces manual attendance registers with a **digital
platform** that allows users to manage student records, mark attendance,
and view attendance history easily.

------------------------------------------------------------------------

# 🚀 Features

-   🔐 Secure Login Authentication
-   👨‍🎓 Student Record Management
-   📅 Quick Attendance Marking
-   📊 Attendance History Tracking
-   🗂 MySQL Database Integration
-   🎨 Clean and Responsive Interface

------------------------------------------------------------------------

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

    git clone https://github.com/Atulkumar58/Attendanceapp.git

## 2️⃣ Move the project to XAMPP directory

Place the folder inside:

    xampp/htdocs/

## 3️⃣ Start XAMPP

Start the following services:

-   Apache
-   MySQL

## 4️⃣ Setup Database

1.  Open **phpMyAdmin**
2.  Create a new database
3.  Import the provided SQL file

## 5️⃣ Run the project

Open your browser and go to:

    http://localhost/Attendanceapp
